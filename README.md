BigBang
========

El repositorio usa `workspaces` al igual que `Vue` o `React`

1. Instalar la librería bigbang

```sh
# Ingresar a la carpeta bigbang
cd packages/bigbang

# Compilar la librería
yarn build

# Ejecutar storybook
yarn storybook
```

2. Probar el demo

```sh
# Ingresar a la carpeta demo
cd sites/demo

# Iniciar la demo
yarn dev
